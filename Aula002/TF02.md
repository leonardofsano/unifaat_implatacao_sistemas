# TF02 - Containerização de Aplicação Web
**Disciplina:** Implementação de Software  
**Curso:** Análise e Desenvolvimento de Sistemas - UniFAAT  
**Aula:** 02 - Virtualização e Containers (Docker)  
**Valor:** 2 pontos  
**Prazo:** 1 semana após a aula


## Objetivo

Demonstrar domínio de containerização com Docker, criação de imagens personalizadas e gerenciamento de containers.


## Tarefa

Containerizar uma aplicação web completa usando Docker, criando imagens otimizadas e implementando boas práticas de containerização.


## Especificações

### Aplicação Web
Desenvolver uma aplicação web simples com:
- **Landing page** de um produto/serviço
- **Página de recursos/features**
- **Página de preços**
- **Página de contato**

### Requisitos Técnicos

**1. Dockerfile Otimizado:**
- Usar imagem base Alpine
- Multi-stage build (se aplicável)
- Instalar apenas dependências necessárias
- Configurar usuário não-root
- Implementar healthcheck

**2. Aplicação Web:**
- HTML5, CSS3 e JavaScript
- Design responsivo
- Pelo menos 4 páginas funcionais
- Formulário de contato funcional (validação frontend)

**3. Configuração Docker:**
- Container deve rodar na porta 80 internamente
- Mapeamento para porta 8080 do host
- Volume para logs (opcional)
- Variáveis de ambiente para configuração

**4. Documentação:**
- README com instruções completas
- Dockerfile comentado
- Docker Compose para facilitar execução

## Dockerfile Exemplo

```dockerfile
# Use imagem Alpine para menor tamanho
FROM nginx:alpine

# Instalar curl para healthcheck
RUN apk add --no-cache curl

# Criar usuário não-root
RUN addgroup -g 1001 -S appgroup && \
    adduser -u 1001 -S appuser -G appgroup

# Copiar arquivos da aplicação
COPY src/ /usr/share/nginx/html/
COPY nginx/nginx.conf /etc/nginx/nginx.conf

# Configurar permissões
RUN chown -R appuser:appgroup /usr/share/nginx/html

# Expor porta
EXPOSE 80

# Healthcheck
HEALTHCHECK --interval=30s --timeout=10s --start-period=5s --retries=3 \
  CMD curl -f http://localhost/ || exit 1

# Usar usuário não-root
USER appuser

# Comando padrão
CMD ["nginx", "-g", "daemon off;"]
```

- Docker Compose instalado

### Opção 1: Docker Compose (Recomendado)
```bash
# Clone o repositório
git clone [URL_DO_SEU_REPO]
cd TF02

# Execute com Docker Compose
docker-compose up -d

# Acesse: http://localhost:8080
```

### Opção 2: Docker Manual
```bash
# Construir a imagem
docker build -t minha-app:v1 .

# Executar o container
docker run -d -p 8080:80 --name minha-app minha-app:v1

# Acesse: http://localhost:8080
```

# Parar e remover

```
